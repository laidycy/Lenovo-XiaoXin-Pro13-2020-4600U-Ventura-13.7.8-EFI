# Lenovo-XiaoXin-Pro13-2020-4600U-Ventura-13.7.8-EFI
联想小新Pro13 2020 锐龙4600U 博通BCM94352 黑苹果 Ventura-13.7.8 文件， 基于黑苹果老吴的config文件修改升级而来，放弃了恢复版和单系统，直接采用单硬盘单系统，硬盘由三星pm981a更换为西数SN750，网卡由RTL8822CE更换为拨通BCM94352，目前除了自动亮度不工作（手动亮度正常），触摸板略微奇怪，其余都正常 <br>
主要配置： <br>
CPU：锐龙R5 4600U <br>
内存：16G <br>
硬盘：~~三星pm981a~~ 西数sn750 500G <br>
网卡：~~RTL8822CE~~ 博通BCM94352  <br>
